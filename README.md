A simple To-Do List application built with HTML, CSS, and JavaScript, featuring dynamic task management and persistent storage.

Features

✅ Add Tasks – Easily add new tasks.

✅ Delete Tasks – Remove tasks individually.

✅ Mark as Completed – Click a task to toggle its completion status.

✅ Clear All – Remove all tasks with a single click.

✅ Persistent Storage – Tasks are saved in localStorage, so they remain after page reload.

✅ Responsive Design – Works on both desktop and mobile devices.

✅ Professional Styling – Smooth hover effects and clean UI.
Installation

1.Clone this repository:

git clone: https://github.com/mdimtiyaz-adv/simple-to-do-app.git

Open index.html in your browser.
Usage

Type a task in the input field and click Add.

Click a task to mark it as completed.

Click Delete to remove a single task.

Click Clear All to remove all tasks.
